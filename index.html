            Cómo publicar en GitHub Pages

            Instrucciones resumidas (también están en README.md).

            1. Crea un repositorio en GitHub llamado <tu-usuario>.github.io.
            2. Inicializa git en esta carpeta, haz commit y empuja al repositorio.
            3. En GitHub → Settings → Pages, selecciona Deploy from a branch y la rama main (root).
